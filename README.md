# Что-то типа охранной системы
Камера снимает и в своем качестве передает фреймы в локалхост на порту 5000.
Раз в час делается клип и сохраняется в **storage**, каждый час проверяются все записи и если есть старше 7 дней - удаляются.
Все конфиги меняются в config.py:
* порт
* дилей проверки на старые записи
* максимальное время хранения
* максимальное время клипа
* сейв мод (можно выключить сохранение, чтобы просто фреймы шли в локалхост)
* индекс камеры (если 0 - используется камера по умолчанию, если текст (например путь к видео) или другие цифры - поведение другое - cv2.VideoCapture)

Далее будет дорабатываться:
* детект объектов (yolo?)
* коннект метео датчиков

## Обязательно
Для хорошего сжимания видео, использовался h264. Для этого пришлось скачать его версию (.dll) 1.8.0 и закинуть в /server/.venv/Scripts - рядом с python.exe короче
Версии:
* python 3.13.2
* pip 26.1.1
* opencv-python 4.13.0.92
* flask 3.1.3
